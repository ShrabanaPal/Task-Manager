# 🗂️ Task Manager Application

## 📌 Project Overview

This is a full-stack Task Manager application that allows users to manage their daily tasks efficiently.
It includes a frontend for user interaction and a backend for handling business logic and data.

---

## 🧩 Tech Stack

* **Frontend:** (e.g., React / HTML / CSS / JavaScript)
* **Backend:** (e.g., Java / Spring Boot / Rest APIs / Spring Data JPA / Lombok / Springboot Started Test / Springboot DevTools)
* **Database:** (e.g., MySQL)

## ⚙️ Setup Instructions

### 🔹 Prerequisites

Make sure you have installed:

* Node.js & npm
* Java (if using Spring Boot)
* IntelliJ IDEA (for backend)
* VS Code (for frontend)

---

### 🚀 Frontend Setup

1. Navigate to frontend folder:

```
cd frontend
```

2. Install dependencies:

```
npm install
```

3. Run the frontend:

```
npm start
```

👉 The frontend will run on: `http://localhost:3000`

---

### 🚀 Backend Setup

1. Navigate to backend folder:

```
cd backend
```

2. Open the project in IntelliJ IDEA

3. Run the main application file
   (e.g., `Application.java` or your main class)

👉 Backend will run on: `http://localhost:8080`

---

## 🔗 API Integration

Ensure the frontend is configured to call backend APIs correctly (e.g., base URL set to `http://localhost:8080`).

## 📌 Assumptions

* The application is designed for single-user usage.
* Basic validation is implemented.
* Backend APIs are assumed to run locally.

## ⚖️ Trade-offs

* Advanced features like authentication are not implemented.
* Error handling is basic and can be improved.
